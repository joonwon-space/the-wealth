# THE WEALTH — Tasks (현재 작업)

이 문서는 **지금 바로 실행할 작업** 목록이다.
`/auto-task`와 `/next-task`가 이 문서에서 작업을 읽는다.
`/discover-tasks`가 이 문서를 갱신한다.

각 항목은 하나의 커밋 단위로 완료 가능한 크기여야 한다.
